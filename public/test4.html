🛠️ 部署配置

环境变量设置

在Deno Deploy或本地运行时设置以下环境变量:

ACCOUNT1_COOKIES="your_account1_cookies_here"
ACCOUNT2_COOKIES="your_account2_cookies_here"
ACCOUNT3_COOKIES="your_account3_cookies_here"
PORT=8000

本地运行

克隆代码后本地运行:

# 设置环境变量
export ACCOUNT1_COOKIES="..."

# 运行服务
deno run --allow-net --allow-read --allow-env main.ts

Deno Deploy

一键部署到Deno Deploy:

1. Fork项目到GitHub
2. 连接到Deno Deploy
3. 设置环境变量
4. 自动部署完成

获取Cookies

在Grok网站登录后，按F12打开开发者工具:

1. 打开Network标签
2. 刷新页面
3. 找到任意请求
4. 复制Cookie请求头完整内容
